Technology matters

Textile is built on open source code, the IPFS protocol, and a mission to improve the future of our data. Are you ready to join us?

How it works

  • Create an account

    Every user gets a unique and secure private key that is yours and never shared.

  • Save pictures

    Every photo a user takes is encrypted with their private key and given a unique fingerprint.

  • Track your memories

    That fingerprint is stored in the user's digital wallet, where they keep track of all the photos they've taken.

  • Sync every photo

    If the user chooses, encrypted versions of the photo are synced across a user's devices or backed up remotely on IPFS.

  • Share safely

    If the user shares the photo in a group thread, the photo is re-encrypted with a shared key and broadcast to thread members.
